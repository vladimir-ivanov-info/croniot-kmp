 # Arquitectura Android Croniot

- Kotlin Multiplatform + Android
- UI en Jetpack Compose
- DI con Koin
- Estructura por módulos feature/*
- core:data, core:domain y core:ui contienen piezas reutilizables
- Seguir Clean Architecture
- No romper compatibilidad entre app, server y capa IoT sin avisarlo
- Priorizar soluciones fáciles de testear
